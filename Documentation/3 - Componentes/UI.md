## 🛠️ Cómo añadir componentes
Usa el comando de `shadcn-ui` para añadir nuevos componentes:

```bash
npx shadcn-ui@latest add [component-name]
```

Luego, recuerda crear su nota correspondiente en `Documentation/3 - Componentes/`.
