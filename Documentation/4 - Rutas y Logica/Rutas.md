# 🛣️ Mapa de Rutas (App Router)

Este proyecto utiliza el **Next.js App Router**, que permite una gestión eficiente de layouts y streaming.

## 📁 Estructura de `src/app`

| Ruta | Archivo | Descripción |
| :--- | :--- | :--- |
| `/` | `page.tsx` | Landing Page principal (Punto de entrada). |
| - | `layout.tsx` | Layout raíz (Fuentes, HTML, Providers). |
| - | `globals.css` | Estilos globales y configuración de Tailwind. |

## 🏗️ Layout Raíz
Ubicación: `src/app/layout.tsx`
Configura las fuentes (Geist Sans/Mono) y el contenedor principal para todas las páginas.
